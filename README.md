Note: delete routes are there for testing purposes.


## Instructions

### Start

Server will listen on port `3000` by default (can be overridden with environment variable `PORT`)
```sh
$ npm start
```


### Test

```sh
$ npm test
```


### Lint

```sh
$ npm run lint
```
